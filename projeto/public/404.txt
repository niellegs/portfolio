
Se você quiser adicionar mais páginas e substituir o uso do operador ternário, uma abordagem mais escalável seria criar um componente separado para cada página e usar o estado para controlar qual componente de página deve ser exibido.

Aqui está um exemplo de como você pode fazer isso:

jsx
Copy code
import React, { useState } from 'react';
import './intro.css';

// Componente da página inicial
const Home = () => {
  return (
    <div>
      <h1>Página Inicial</h1>
      {/* Conteúdo da página inicial */}
    </div>
  );
};

// Componente da página "Sobre Mim"
const AboutMe = () => {
  return (
    <div>
      <h1>Sobre Mim</h1>
      {/* Conteúdo da página "Sobre Mim" */}
    </div>
  );
};

// Componente da página 404
const NotFound = () => {
  return (
    <div>
      <h1>Are you lost babygirl?</h1>
      {/* Conteúdo da página 404 */}
    </div>
  );
};

function Intro() {
  const [currentPage, setCurrentPage] = useState('home');

  const handlePageChange = (page) => {
    setCurrentPage(page);
  };

  let pageContent;

  if (currentPage === 'home') {
    pageContent = <Home />;
  } else if (currentPage === 'about') {
    pageContent = <AboutMe />;
  } else {
    pageContent = <NotFound />;
  }

  return (
    <div className="intro-container">
      <nav>
        <ul>
          <li onClick={() => handlePageChange('home')}>HOME</li>
          <li onClick={() => handlePageChange('about')}>SOBRE MIM</li>
          {/* Adicione mais opções de menu para outras páginas */}
        </ul>
      </nav>

      <div id="output">{pageContent}</div>
    </div>
  );
}

export default Intro;
Neste exemplo, foram adicionados os componentes Home, AboutMe e NotFound para representar as diferentes páginas. O estado currentPage controla qual página está sendo exibida. Quando ocorre um clique em um item do menu, a função handlePageChange é chamada para atualizar o estado currentPage com o valor da página correspondente.

Dentro do componente Intro, um bloco if-else é usado para determinar qual componente de página deve ser renderizado com base no valor do estado currentPage. Assim, você pode adicionar mais opções de menu e páginas, criando novos componentes correspondentes.

Espero que isso ajude! Lembre-se de importar corretamente os componentes de página e fazer as modificações necessárias nos conteúdos individuais de cada página de acordo com suas necessidades.
